What is Dissipative Asymmetry?

Many physical systems are made of independent pieces that can be in two states: on or off (excited or ground, 1 or 0). When you read these pieces, errors happen — but not equally in both directions.

Example: imagine a sensor with 100 pixels.

p_d — dominant error

An "on" pixel is misread as "off." This is the common error — things naturally turn off (charge leaks, qubits decay, DNA degrades).

Example: p_d = 0.04 (4%)

p_e — excitation error

An "off" pixel is misread as "on." This is the rare error — things don't spontaneously turn on without energy.

Example: p_e = 0.001 (0.1%)

The coefficient α

From these two error rates, you compute a single number:

α = ln[(1 − p_d) / (1 − p_e)]

With the example above: α = ln[0.96 / 0.999] = −0.0398

This number tells you how different the two error directions are.

What is k?

k is how many elements are "on" (excited) in your system.

ln[P(k) / P(0)] = k × α

Every extra "on" element changes the readout probability by the same amount α.

What is F?

F represents the operating conditions: temperature, voltage, humidity.

The key property: F cancels exactly in the ratio P(k)/P(0). If you change the temperature, total errors change, but α stays the same.

α is a property of the system, not of the conditions.

Three things you can do with α

1. Measure once

Compute α from p_d and p_e. This single number characterizes your system. No need to measure again unless the system physically changes.

2. Estimate

For any k, the readout accuracy relative to k=0 is e^(k×α). Just multiply k by α. No extra measurements needed.

3. Detect degradation

If α changes over time, the system has degraded. If α is stable but errors increase, only the environment changed.

Characterization

Measure p_d and p_e once. Get α — one number that describes your system's intrinsic asymmetry, independent of operating conditions.

Monitoring

If α changes over time, the system has physically degraded. If α is stable but errors increase, only the environment changed.

Comparison

Two devices, different manufacturers, different conditions. Compare α — because α does not depend on conditions. Same α = same intrinsic quality.
